Third-Party Payer Reimbursement Analysis

Project Overview

This healthcare analytics project examines third-party payer reimbursement trends over a 10-year period to identify patterns in insurance performance, reimbursement efficiency, and revenue cycle optimization opportunities.

Using PostgreSQL and Tableau, I cleaned and analyzed payer data to uncover reimbursement disparities across insurance providers and visualize financial trends impacting healthcare operations.

Business Problem

Healthcare organizations rely heavily on accurate reimbursement and payer performance analysis to maintain financial stability. This project was designed to explore how payer mix and reimbursement trends can affect long-term revenue cycle performance.

Tools & Technologies

PostgreSQL Tableau Healthcare Analytics Data Cleaning Data Visualization Revenue Cycle Analysis

Key Insights

  • Identified payer reimbursement trends across a 10-year financial dataset.
  • Compared high-performing and low-performing insurance reimbursement patterns.
  • Developed interactive Tableau dashboards to improve financial trend visibility.
  • Cleaned and transformed raw healthcare financial data using PostgreSQL.
  • Created visual reporting tools to support operational and revenue cycle decision-making.

Future Improvements

  • Add predictive reimbursement forecasting models.
  • Expand analysis to include patient demographics and claim denial trends.
  • Integrate real-time dashboard filtering and KPI monitoring.
