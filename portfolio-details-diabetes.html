Healthcare Efficiency & Risk Analysis

Project Overview

Patient Readmission Risk Optimization

An analysis of 100,000+ hospital encounters to identify clinical predictors for patient readmission using PostgreSQL for data engineering and Tableau for visualization.

Tableau Dashboard: Readmission Risk Factors

The Challenge

Processing a high-volume, "messy" healthcare dataset required significant cleaning, handling vertical data formats, and type-casting in PostgreSQL to ensure data integrity.

Clinical Approach

Leveraging my Doctor of Chiropractic background, I engineering clinical features—such as identifying emergency vs. elective admission impacts—to ensure the model's findings remained medically relevant.

Project Tech Stack

  • Database: PostgreSQL
  • Visuals: Tableau
  • Version Control: GitHub
  • Tools: VS Code
